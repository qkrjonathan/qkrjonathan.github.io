Introduction

Back to top

Official support for Windows 10 is ending on 14 October 2025. Obviously this doesn't mean your existing Windows 10 computer just magically explodes or crashes on that date Y2K-style; but simply put, Windows will simply stop caring about that version. No more security updates, no more technical support. You're on your own.



Which leaves you seemingly with a couple options, which all sound horrible.

The first is switching to Windows 11. The fact that some machines running Windows 10 can't even meet the hardware requirements for the new version says more than enough. Thankfully(?) my laptop was not one of those machines. When I eventually reneged to the endless barrage of "reminders" to update to Windows 11, however, my Bluetooth headphones refused to connect, and my already dying trackpad (which is my fault, I admit) gladly took the opportunity to simply cease to function.

The other is ... throwing your laptop away? Or as Microsoft sugarcoats it, "to leverage trade-in and recycling programs." Recycling is fine and all, but it doesn't change the fact that if my computer were to not meet the requirements, or I simply did not like Windows 11, then my own computer, that is currently in my possession, and that I bought with my own money, and that clearly still works???? basically becomes waste.

But there has always been a secret third option. Well, not so secret, because so many people already use it; and even more people are probably using it ever since PewDiePie made his YouTube video about it, which has amassed over 6.4 million views at the time of this post.

But yes. It's switching to Linux.

Linux has somewhat of a reputation, partially for being a niche thing that only nerds use, and partially for being the thing that annoying people use. (These are not mutually exclusive.)

Whether I myself, having run some distribution of Linux as my daily driver for almost a year now, have become one of those said annoying people is a question I will decidedly ignore. In any case, the former is definitively untrue. Android runs on Linux. So does Instagram, Facebook, Twitter, YouTube, the New York Stock Exchange.

And so should you. Potentially. Perchance.

Maybe you feel a bit intimidated by the fact that you're running, to some extent, the same operating system as the Pentagon. (Which is probably not true.) But for you, perhaps a more casual user, Linux could look as simple as this:


Courtesy Linux Mint

Clean, sleek, and kind of looks like Windows?

Exactly. This is Linux Mint. Here's why you should get it.

Why you should get it

  • Basically zero setup. You follow the instructions on a website, no code, no nothing. You do need a USB stick.

  • It's free. And there's no catch. Unlike with some companies.

  • It's open-source, which is no guarantee of anything, but it does mean if the developer tried to do something weird with it someone could actually notice. Unlike with some companies.

  • There's much less bloatware, meaning even older machines can run faster (and Mint is one of the more bloated distros).

  • No annoying notifications, e.g. about Xbox Game Pass which no one gives a fuck about,

  • Basically most of the apps you need for work/life just work, out of the box (Slack, Discord, browsers, VPNs, text editors),

  • You can try it as long as you want before you actually commit to it. Linux Mint can boot from a USB stick, and it will stay within that USB stick until you explicitly agree to have it take over your laptop (or dual-boot with Windows, see under "Further considerations.")

Why you shouldn't get it

  • You're forced to rely on Microsoft tools, e.g. for work (Outlook, Teams, Edge 👎 (why)). This is assuming you don't have a separate work laptop that you could rely on for such software.

  • You play any games that have no anti-cheat support for Linux, current or planned. That includes Valorant, Apex, Battlefield 2042, Fortnite ... You can see the full list here. If you consider yourself a hardcore PC gamer then Linux can feel pretty restricted. But, the good news is there already exists a software called Proton that makes (many but not all) Windows games run on Linux. Basically you can assume that if it's Steam Deck Verified, it runs on Linux.

  • You rely on ANY Adobe creative software (Photoshop, Premiere, Illustrator, etc.) ... but considering just how predatory Adobe's services have been becoming, maybe this could be your opportunity to escape its ecosystem entirely? For example, here's a YouTube video on making videos entirely in Linux.

Further considerations

  • If you're a coding person you will need to build back up all those environments again. (IDEs, cloned repos from GitHub, Git CLI config) (get ready to read forum posts buddy) (but also if you're a coding person you'd perhaps be more predisposed to switching to Linux as your daily driver anyway) (think about it)

  • Installing apps takes place not through those installation wizards like in Windows but through packages that you can either get in the app-store-like Software Manager, or in the command line via sudo apt-get install [name]

  • There is no Microsoft Office (as implied earlier), and the equivalent open-source LibreOffice suit is installed by default. I will admit, they are a bit chaotic and annoying compared to the Microsoft apps, but they're still usable, which is ultimately all that matters. Nothing is stopping you from using Google Docs etc. although there are separate reasons for avoiding those.

  • If you take online-proctored tests e.g. via Pearson VUE for any reason (such as getting certifications for your career), those tend to only work on Windows. And virtual machines don't do the trick (anymore) either. Source: I had to deal with this after I'd already switched to Linux prior to taking my Splunk Core Certified User exam.

  • If, unfortunately but realistically, you still absolutely need Windows in your life (Adobe, MS Office and online exams are among the strongest reasons in my own personal life, at least), you have options!!!

    • The easiest (and free): You can always dual-boot Linux Mint and Windows on the same machine. (Yes, your computer can have more than one operating system on the same hard drive.) More information here if you're curious.

      But important note: WINDOWS MUST BE INSTALLED FIRST!!!!!, i.e. do not try to re-install Windows after you've already wiped your entire drive for Linux Mint. I have an entire blog post on why this was an absolute nightmare.

    • Less ideal would be to have an entirely separate laptop that's exclusively Windows, and your current one exclusively Linux. Hardware-wise it's the most "clean," if that's important for your brain (no fiddling with drives); but there are only a handful of scenarios where this is realistic, e.g. if you're in a family that has more than one computer to spare around the house.

Minimum system requirements

This is taken directly from the Linux Mint installation guide. (It is pretty safe to assume a laptop or PC meets these requirements.)

  • 2GB RAM (4GB recommended for a comfortable usage).

  • 20GB of disk space (100GB recommended).

  • 1024×768 resolution (on lower resolutions, press ALT to drag windows with the mouse if they don’t fit in the screen).

For comparison, you can expect a fairly decent laptop from Walmart to come with 8-32GB RAM, 256+ GB storage, and 1920x1080 (1080p HD) resolution.

Introdução

Voltar acima

O apoio oficial do Windows 10 vai acabar no dia 14 de outubre 2025. Claro que isto não significa que o seu notebook existente com Windows 10 magicamente explode ou falha nessa data exata, mas, falando simplesmente, o Microsoft simplesmente vai deixar de se importar dessa versão. Nem mais atualizações de segurança, nem mais apoio tecnical. Você será sozinho.



Depois disso, você tem agora duas opções, das quais ambos parecem horriveis.

A primeira é mudar para o Windows 11. O fato de que algumas máquinas que agora mesmo estão usando Windows 10 nem podem satisfazer os requisitos de hardware para a nova versão diz mais do que o suficiente. Felizmente(?) meu notebook não foi uma dessas máquinas. Quando finalmente cedi ao monte das "lembretes" para atualizar ao Windows 11, embora, meus fones Bluetooth refusou a conectar-se, e meu trackpad que já estava morrendo (que é a culpa minha — isso eu admito) aproveitou com prazer a oportunidade para simplesmente deixar de funcionar.

A outra é ... jogar fora seu notebook inteiro?? Ou, como o Microsoft expressa mais suavemente, "aproveitar programas de troca ou reciclagem." É bom reciclar, claro, mas isso não muda o fato de que se meu computador não satisfazesse os requisitos, ous e eu simplesmente não gostasse do Windows 11, então meu próprio computador, que atualmente está na minha posse, e que comprei com meu próprio dinheiro, e que claramente ainda funciona???? básicamente fica lixo.

Mas sempre tem sido uma terçeira opçao secreta. Bem, não tão secreta, porque tanta gente já o usa; e ainda mais pessoas provavelmente estão o usando após o PewDiePie fez seu vídeo sobre ele, que acumulou mais de 6.4 milhões visualizações na época deste post.

Mas sim. É mudar para o Linux.

Olha, já sei que o Linux tem uma reputaçãozinha, em parte para ser uma coisa de nicho que só os nerds usam, e em parte para ser a coisa que as pessoas irritantes usam. (Essas não são mutuamente exclusivas.)

A questão de se eu mesmo, tendo usado alguma distribuição do Linux como meu daily driver há quase um ano, já me tornei uma dessas pessoas irritantes, é uma que claramente vou ignorar. De qualquer forma, a primeira é definitivamente errada. O Android funciona em Linux. O mesmo com o Instagram, o Facebook, o Twitter, o YouTube, a Bolsa de Valores de Nova Iorque.

E será o mesmo com você? Eu acho que deve, sim.

Talvez você tenha um pouco de medo pelo fato de que você executaria, até um ponto, o sistema de operação igual como o Pentágono. (Isso provavelmente não é a verdade.) Mas para você, o Linux poderia parecer tão simples como isso:


Foto do Linux Mint

Limpo, elegante, e parece um pouco como o Windows?

É isso. Este é o Linux Mint. Aquí está porque você deve adquiri-lo.

Porque você deve adquiri-lo

  • Basicamente nenhuma configuração. É só seguir as instruções num sítio do web, sem qualquer código, sem nada. Mas o que voce precisa é um pendrive.

  • É grátis. E não há nenhuma condição. Ao contrário de algumas empresas.

  • É de código aberto, que, claro, não garante nada, mas significa que se o desenvolvidor tentasse fazer algo estranho com ele, alguem realmente poderia perceber. Ao contrário de algumas empresas.

  • Sem notificações irritantes, por exemplo sobre a porra do Xbox Game Pass de que ninguém se importa.

  • Pode experimentar por qualquer tempo que você quer antes de dedicar-se a ele.

    O Linux Mint pode executar-se desde um pendrive, e ficará assim até você explicitamente concorda a ele assumir o controle do seu notebook (ou dual-boot com o Windows, veja sob "Outras considerações").

Porque você NÂO deve adquiri-lo

  • Você é obrigado a depender de ferramentas do Microsoft, e.g. para trabalho (Outlook, Teams, Edge 👎). Isso é assumindo que você não tem um outro notebook para trabalho de que pode depender para software assim.

  • Você joga qualquer jogo que não tem apoio anti-cheat para Linux, atual ou planeado. Isso inclui Valorant, Apex, Battlefield 2042, Fortnite ... Pode ver a lista inteira aqui. Se vpcê considera-se um hardcore PC gamer, então Linux pode parecer bem restrito. Mas, as boas noticias são que já existe um software chamado Proton que faz (muitos, mas nem todos) os jogos no Windows executar-se no Linux. Basicamente pode assumir que se for verificado para Steam Deck, executa-se no Linux.

  • Você depende de qualquer software criativo do Adobe (tipo Photoshop, Premiere, Illustrator, etc.) já que eles não existem no Linux e nunca vâo existir ... mas considerando só quão predatório os serviços do Adobe tem sido tornando-se, talvez isso poderia sua oportunidade para escapar o ecossistema dele completamente? Por exemplo, aquí é um vídeo no YouTube sobre fazer vídeos inteiramente no Linux.

Outras considerações

  • Se você for um pessoa que gosta de programação, você terá que refazer todos os seus ambientes de novo. (IDEs, repositórios clonados do GitHub, configuração do Git CLI)

  • Instala-se aplicações não pelos assistentes de instalação como se vê no Windows, mas por pacotes que você pode obter no Software Manager (que funciona igual como o app store), ou pelo command line via sudo apt-get install [nome].

  • Não há Microsoft Office, claro, e o equivalente de código aberto, LibreOffice, é instalado por padrão. Admito que pode ser um pouco caótico e chato para usar, comparado com os apps do Microsoft, mas ainda são utilizáveis, qual é ultimamente tudo que se importa. Tem nada impedindo você de usar os apps do Google, embora tem outras razões por nem usar eles.

  • Se você fazer provas supervisionado online, tipo pelo Pearson VUE, por qualquer razão (tipo obter certificados para sua carreira), no usual esses só funcionam no Windows. E não dá para usar as máquinas virtuais, também. Digo isso porque eu já tinha problemas com o meu exame de Splunk Core Certified User após já mudar para o Linux.

  • Se, infelizmente mas realmente, ainda absolutamente precisa do Windows na sua vida (Adobe, Microsoft Office e os exames online são umas das razões mais fortes na minha vida pessoal, pelo menos), você tem opções!!!

    • A mais fácil (e gratuita): Sempre pode fazer dual-boot do Linux Mint e do Windows na mesma máquina. (Sim, seu computador pode ter mais de um sistema de operação no mesmo disco rígido.) Há mais informações aqui se você for curioso.

      Mas uma nota importante: O WINDOWS DEVE SER INSTALADO PRIMEIRO!!! Ou seja, nem tente de reinstalar o Windows após já apagar o seu disco rígido inteiro para o Linux Mint. Tenho um blog post inteiro sobre porque esse foi um verdadeiro pesadelo.
